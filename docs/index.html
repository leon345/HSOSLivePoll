Einführung

Willkommen beim LivePoll Umfragentool! Diese Anwendung ermöglicht es Ihnen, interaktive Umfragen zu erstellen, durchzuführen und die Ergebnisse in Echtzeit zu verfolgen.

Hauptfunktionen:

  • Erstellen von Umfragen mit verschiedenen Fragetypen
  • Echtzeit-Abstimmung über QR-Codes oder Kurzlinks
  • Live-Präsentation der Ergebnisse
  • Benutzerfreundliche Oberfläche
  • Sichere Anmeldung über Microsoft Azure AD

Landing Page

Die Landing Page ist der erste Kontaktpunkt für alle Benutzer des Umfragentools.


Landing Page mit Logo und Hauptfunktionen

Verfügbare Aktionen:

  • An einer Umfrage teilnehmen: Geben Sie den Kurzcode ein und klicken Sie auf den Pfeil
  • Neue Umfrage erstellen: Klicken Sie auf "Umfrage erstellen" (erfordert Anmeldung)

Tipp

Sie können direkt über den Kurzcode an einer Umfrage teilnehmen, ohne sich anmelden zu müssen.

Umfrage erstellen

Nach der Anmeldung können Sie neue Umfragen erstellen und verwalten.

Schritte zum Erstellen einer Umfrage:

  1. Melden Sie sich mit Ihrem Microsoft-Konto an
  2. Klicken Sie auf "Neue Umfrage erstellen" im Dashboard
  3. Füllen Sie die Umfrage-Details aus
  4. Klicken Sie auf "Umfrage erstellen"

Formular zum Erstellen einer neuen Umfrage

Wichtige Felder:

  • Titel: Kurzer, beschreibender Name für die Umfrage
  • Optionen: Antwortmöglichkeiten (mindestens 2, maximal 10)
  • Typ: Wählen Sie zwischen "Single Choice" und "Multiple Choice"

Umfrage-Typen

Das System unterstützt zwei verschiedene Umfragetypen:

Single Choice

Teilnehmer können nur eine Antwort auswählen. Ideal für Entscheidungsfindung und Meinungsumfragen.

  • Eine Stimme pro Teilnehmer
  • Eindeutige Ergebnisse
  • Perfekt für Abstimmungen

Multiple Choice

Teilnehmer können mehrere Antworten auswählen. Gut für Umfragen mit mehreren Präferenzen.

  • Mehrere Stimmen möglich
  • Flexiblere Antworten
  • Ideal für Präferenzumfragen

Wichtig

Der Umfragetyp kann nach der Erstellung nicht mehr geändert werden. Wählen Sie sorgfältig aus!

Dashboard

Das Dashboard ist Ihr zentraler Verwaltungsbereich für alle Umfragen.

Wichtig

Wenn Sie versuchen, auf das Dashboard zuzugreifen, werden Sie automatisch zur Anmeldeseite weitergeleitet, falls Sie noch nicht angemeldet sind.


Übersicht aller Umfragen und Verwaltungsfunktionen

Dashboard-Funktionen:

  • Aktive Umfragen: Übersicht aller laufenden Umfragen
  • Letzte Umfragen: Schnellzugriff auf kürzlich erstellte Umfragen
  • Neue Umfrage: Button zum Erstellen einer neuen Umfrage
  • Umfrage-Details: Klicken Sie auf eine Umfrage für detaillierte Informationen

Anmeldung mit Microsoft

Das Umfragentool verwendet Microsoft Azure Active Directory für eine sichere Authentifizierung.

Wichtig

Für die Anmeldung können in der Testversion nur Test-Accounts verwendet werden. Persönliche Microsoft-Konten werden nicht unterstützt.

Anmeldeprozess:

  1. Klicken Sie auf "Anmelden" in der oberen rechten Ecke
  2. Sie werden zur Microsoft-Anmeldeseite weitergeleitet
  3. Geben Sie Ihre persönlichen Microsoft-Anmeldedaten ein
  4. Bestätigen Sie die Anmeldung (falls erforderlich)
  5. Sie werden zurück zum Umfragentool geleitet

Microsoft Azure AD Anmeldeseite

  • Sichere Verbindung über HTTPS
  • Keine Passwörter werden im System gespeichert
  • Automatische Abmeldung bei Inaktivität

Unterstützte Kontotypen in der Testversion:

  • Unterstützt: Test-Accounts für Entwicklungs- und Testzwecke oder eingetragene Microsoft-Konten
  • Nicht unterstützt: Persönliche Microsoft-Konten (@outlook.com, @hotmail.com, @live.com), Unternehmens-Konten von Microsoft 365, Azure AD, oder andere Organisations-Konten

Abstimmung

Teilnehmer können über verschiedene Wege an Umfragen teilnehmen.

Teilnahme-Methoden:

QR-Code

Scannen Sie den QR-Code mit Ihrem Smartphone

Kurzlink

Geben Sie den Kurzcode auf der Landing Page ein

Direkter Link

Öffnen Sie den generierten Link in einem Browser


Abstimmungsseite mit Frage und Antwortoptionen

Abstimmungsprozess:

  1. Öffnen Sie die Abstimmungsseite
  2. Lesen Sie die Frage und Antwortoptionen
  3. Wählen Sie Ihre Antwort(en) aus
  4. Klicken Sie auf "Abstimmen"

Tipp

Nach der Abstimmung können Sie die aktuellen Ergebnisse sehen. Ihre Stimme wird anonym behandelt.

Präsentationsmodus

Der Präsentationsmodus zeigt die Umfrageergebnisse in Echtzeit an.


Live-Anzeige der Umfrageergebnisse

Präsentationsfunktionen:

  • Live-Updates: Ergebnisse werden in Echtzeit aktualisiert
  • Grafische Darstellung: Balkendiagramme und Prozentangaben
  • Responsive Design: Funktioniert auf allen Bildschirmgrößen
  • Teilnehmerzahl: Anzeige der Gesamtanzahl der Stimmen

Verwendung:

Öffnen Sie den Präsentationsmodus über das Dashboard und projizieren Sie ihn auf einen großen Bildschirm oder Beamer. Die Ergebnisse werden automatisch aktualisiert, wenn neue Stimmen eingehen.

Importieren & Exportieren

Das LivePoll Umfragentool bietet umfangreiche Funktionen zum Importieren und Exportieren von Umfragen und Ergebnissen.

Umfragen exportieren

Sie können bestehende Umfragen als XML-Vorlage exportieren, um sie später wiederzuverwenden oder mit anderen zu teilen.

Export-Funktionen:

  • XML-Export: Exportieren Sie Umfragen als XML-Datei
  • Wiederverwendung: Importieren Sie exportierte Umfragen als Vorlage
  • Teilen: Teilen Sie Umfragevorlagen mit Kollegen
  • Backup: Sichern Sie Ihre Umfragen lokal

Umfragen importieren

Importieren Sie zuvor exportierte Umfragen oder Vorlagen, um schnell neue Umfragen zu erstellen.

Import-Prozess:

  1. Wählen Sie "Neue Umfrage erstellen" im Dashboard
  2. Klicken Sie auf "XML importieren"
  3. Wählen Sie eine XML-Datei aus
  4. Überprüfen Sie die importierten Daten
  5. Passen Sie bei Bedarf Titel oder Optionen an
  6. Erstellen Sie die Umfrage

Unterstützte Formate:

  • XML-Dateien: Standardformat für Umfragevorlagen
  • Kompatibilität: Alle exportierten XML-Dateien können wieder importiert werden
  • Validierung: Automatische Überprüfung der Import-Daten

Ergebnisse exportieren

Exportieren Sie Umfrageergebnisse für weitere Analysen oder Berichte.

Export-Optionen:

  • CSV-Format: Kompatibel mit Excel, Google Sheets und anderen Tabellenkalkulationsprogrammen
  • Vollständige Daten: Alle Abstimmungen mit Zeitstempel
  • Statistiken: Zusammenfassende Ergebnisse und Prozentangaben
  • Offline-Analyse: Arbeiten Sie mit den Daten außerhalb des Systems

Häufige Fragen

Kann ich eine Umfrage nach der Erstellung bearbeiten?

Nein, die Umfrage muss neu erstellt werden. Man kann auch die Umfrage als XML-Vorlage speichern und dann eine neue Umfrage mit den gleichen Einstellungen erstellen.

Warum ist die Umfrage nicht aktiv?

Umfragen müssen erst aktiviert werden, bevor sie für Teilnehmer sichtbar sind.

Können Teilnehmer mehrfach abstimmen?

Es kann eingestellt werden, dass ein Teilnehmer nur einmal abstimmen kann. Standardmäßig ist diese Einstellung deaktiviert.

Kann ich die Ergebnisse exportieren?

Ja, die Ergebnisse können exportiert werden als CSV Datei
